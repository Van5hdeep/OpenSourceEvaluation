This dashboard uses the following public APIs:

1. Geocoding: [OpenWeather Geocoding API](https://openweathermap.org/api/geocoding-api) * Used to convert a city name into geographic coordinates (latitude and longitude).

2. Weather: [Open-Meteo API](https://open-meteo.com/) * Used to fetch the current temperature and the 24-hour temperature forecast.

3. Air Quality: [World Air Quality Index (WAQI) Project API](https://aqicn.org/api/) * Used to fetch the real-time Air Quality Index (AQI).

4. Traffic: [TomTom Traffic Incidents API](https://developer.tomtom.com/traffic-api/traffic-api-documentation-traffic-incidents/incident-details) * Used to fetch the number of live traffic incidents.
